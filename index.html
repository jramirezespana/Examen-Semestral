¿Qué es una referencia por clase?

Una referencia por clase es una categoria en donde metemos uno o más elementos de una página web la cual podemos utilizar para ajustar características de las mismas

¿Qué es una referencia por palabra clave?

Una referencia por palabra clave es una categoría en donde utilizamos una palabra clave para referenciar uno o más elementos para editar sus características. Es mucho más cerrado a comparación de la referencia por clase.

¿Qué es una referencia por ID?

Una referencia por ID es una categoría donde utilizamos un sólo elemento identificado por un ID específico (numerico) para cambiar sus características.
